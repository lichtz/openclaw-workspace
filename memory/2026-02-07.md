# OpenClaw Web Chat - 项目规划

## 功能目标
- Web聊天界面
- 对接OpenClaw API
- 支持发送消息、接收回复
- 支持语音（调用TTS）
- 实时WebSocket通信

## 技术栈
- 前端：HTML + CSS + JavaScript（简单版）
- 后端：Node.js + Express + WebSocket
- AI对接：OpenClaw HTTP API

## 项目结构

```
openclaw-web-chat/
├── public/
│   ├── index.html      # 主页面
│   ├── style.css       # 样式
│   └── app.js          # 前端逻辑
├── server.js           # 后端服务
├── package.json        # 依赖
└── README.md
```

## 对接OpenClaw

### 方式1：HTTP API
```
POST http://127.0.0.1:18789/message
{
  "channel": "qqbot",
  "message": "用户输入"
}
```

### 方式2：WebSocket
```
ws://127.0.0.1:18789
```
